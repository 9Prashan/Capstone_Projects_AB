# Capstone Projects AB

This repository contains the capstone projects completed as part of AlmaBetter.

## About

This is the repository that contains all the capstone project work for AlmaBetter.

These capstone projects were created to practice core programming and problem-solving skills through hands-on notebook-based exercises. The work focuses on building clean Python solutions, understanding logic step by step, and applying concepts in a practical way.

## Concepts Used

- Python basics such as variables, data types, operators, and expressions
- Conditional statements for decision-making
- Loops and iterations for repeated tasks
- Functions for reusable and organized code
- Lists, tuples, sets, and dictionaries for data handling
- String operations including splitting, replacing, slicing, and formatting
- Object-oriented programming using classes and methods
- Exception handling for safer and more robust programs
- Basic algorithmic thinking, including counting, filtering, searching, and aggregation
- Writing code in a structured format suitable for Jupyter Notebook and Google Colab

## Learning Focus

The projects were designed to strengthen:

- Logical thinking and step-by-step problem solving
- Clean and readable Python coding style
- Use of built-in Python tools and methods
- Understanding of simple real-world coding scenarios
- Confidence in solving beginner-to-intermediate programming questions

## Files

- `Capstone_Project_Introduction_to_Computer_Programming.ipynb`: original notebook
- `CodeStorm_Final_Submission.ipynb`: completed submission notebook

## Run

Open the notebook in Jupyter or Google Colab and run the cells in order.
